alarm folder Internal structure

alarm_app/
│
├─ start.py        ← 最初に起動されるランチャー（PyInstallerでexe化）
│
├─ gui.py
├─ cui.py
│
├─ alarm_manager.py
├─ alarm_model.py
├─ utils.py        ← config管理もここ
├─ constants.py
│
└─ config.json     ← 起動モード記録

 ├─ settings.json   ←（将来拡張用）



gui.py
│
├── import いろいろ
│
├── # Window位置ユーティリティ（クラス外）
│   get_app_dir()
│   get_window_pos_file()
│   load_pos_json()
│   save_pos_json()
│
└── class AlarmGUI:
      ├── __init__()
      ├── save_window_position()
      ├── load_window_position()
      ├── open_settings_window()
      └── ...（メインの処理）



メインウインド（親①）
　└─ 設定ウインド（親②）
　　　├─ 曜日選択ウインド（子）
　　　├─ カレンダー（子）
　　　├─ 時刻ピッカー（子）
　　　└─ カスタム設定（子）
