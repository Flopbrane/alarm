# ★★現在、作成しているアラームについて★★

## ★画面表示部分

### 1．メイン画面

- 年月日(曜日)
- 現在時刻
- 次のアラーム(あと、h時間m分)
- ストップボタン・スヌーズボタン
(スヌーズ中は「スヌーズ：あとn回、あとm分」表示と画面色変更)

### 2．アラーム一覧・新規アラーム登録画面

★上部に登録されているアラーム一覧表示(表形式)
**IDは int ID から UUID に変更**
COLUMN_LABELS: Final[dict[str, str]] = {

以下のような選択肢が存在するドロップダウンリストが出ます

#### 内部値 → 表示（UI表示用）

### 3.カスタムウインド

- 月の第n週指定(複数曜日指定可)
- 毎週n週おき(複数曜日指定可)
＜上記のどちらかを選んでから、
下部に記述されている項目の"どちらか"を選択、編集、自動保存＞
- "第1週～第5週まで、表示されている物から選択"(複数曜日指定可)
- "毎週～5週おき"のリストを選択"(複数曜日指定可)

### 4.曜日ウインド

"月～日まで記述されていて、チェックで選択"

### 5.時刻ウインド

"HH時MM分"の選択入力

### 6.カレンダウインド

上部に"西暦年"、"月"セレクト
その下に、1日単位で、クリックできるボタンを用意して、
選択、設定で、編集、自動保存

**＜気を付けていること＞**
**「なぜ、小さなウインドで選択させるのか？」**
→人が手入力する場合、打ち間違いや、「要求している形式」じゃないことも多いので、
その部分のエラートラップ減らし・エラーによる誤作動を極力無くすため

## ★新規アラーム登録部分

上記の編集方法と同じで、そのを部分をクリックすると、必要なウインドが立ち上がる仕組み。

## 設定ウインドウのアラーム一覧表カラム名変換

COLUMN_LABELS: Final[dict[str, str]] = {
    "id": "ID", # 一覧表示用ID(実稼働時は削除予定)
    "name": "アラーム名",
    "date": "日付",
    "time": "時刻",
    "repeat": "繰返し",
    "weekday": "曜日",
    "enabled": "有効/無効",
    "skip_holiday": "祝日スキップ",
    "duration_time": "再生分数",
    "snooze_limit": "スヌーズ回数上限",
    "end_at": "アラーム有効期限終了日時",
    "custom_desc": "詳細設定",
}

この部分に記述されている項目のセル部分をクリックすると、
編集と同じ様な、小ウインドが出現して、
一番下の"登録ボタン"をクリックすると、新規データとして、アラーム一覧表に反映、自動保存

以上が、画面で表示、選択、編集、変更すると、バックグラウンドで自動保存します。

現在、未だに調整中なので、実働状態に入った場合、
一覧表示では、IDは隠すようにして、「アラーム名」が最初の項目になる予定です。
(IDは内部値(表示されない値)になります)
